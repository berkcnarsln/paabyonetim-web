# BAP Yönetim 🏢

Site yönetim paneli — yöneticiler ve daire sahipleri için.

## Kurulum

```bash
npm install
npm run dev
```

## Demo Hesaplar

| Rol | E-posta | Şifre |
|---|---|---|
| Yönetici | yonetici@bap.com | 1234 |
| Daire Sahibi | daire@bap.com | 1234 |

## Özellikler

- 🔐 Rol tabanlı giriş (Yönetici / Daire Sahibi)
- 💰 Aidat takibi
- 🏠 Daire yönetimi
- 📣 Duyuru sistemi
- 🔧 Arıza bildirimi
- 📊 Gider raporları

## Teknoloji

- React 18 + Vite
- Supabase (backend - yakında)
